Tag p

Atividade sobre os tipos de tag em html, o primeiro exemplo é o "p" onde criamos uma linha de texto na pagina
exemplo: ''O deitroit psitons é o melhor time da nba''

tag h

um outro exemplo que podemos ter é o "h1" que é uma tag de titulo, onde se adicionarmos um número de 1 a 6 depois da letra "h" podemos escolher qual o tamanho do texto.
um exemplo dessa tag sendo utilizada é em todos os titulos desse documento.

tag hr

uma tag também muito utlizada é a hr que faz uma linha podendo dividir os assuntos falados no texto

tag break

A tag break ou br é utilziada quando desejamos pular uma linha durante o decorrer do texto
nesse texto utilizei o break para quebrar a linha de texto antes de acabar o espaço na linha.

tag adress

A tag adress serve para deixarmos o texto em italico.

cade cunnigham joga mais que o sga

tag bold

A tag bold é utilizada para adicionarmos um texto em negrito
esse texto esta em negrito.

tag !
